<h2>Описание проекта</h2>
Суть проекта заключается в том, что веб-приложение является интерактивным информационным справочником. Приложение имеет несколько основных функций:
<ul>
	<ol>Пользователь вводит название продукта и приложение выводит ему сводную информацию о продукте.</ol>
	<ol>Пользователь вводит название блюда и приложение выводит ему сводную информацию о блюде.</ol>
	<ol>Пользователь может составить свою диету на неделю или выбрать одну из предложенных.</ol>
</ul>
